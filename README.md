## Installation
Install dependencies:
```
conda install -c plotly plotly
```

Clone the repository:
```
git clone https://github.com/AlexanderBackis/mg_analysis_notebook.git
```

## Execution
Navigate to mg_analysis_notebook and enter:
```
jupyter notebook
```
